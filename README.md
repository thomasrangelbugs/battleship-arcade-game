# Battleship Arcade Game

> Jogo web de batalha naval tatica com niveis, municao limitada e interface arcade.

## Sobre o projeto

Jogo de batalha naval desenvolvido em JavaScript puro para navegador. O jogador posiciona navios, dispara contra o tabuleiro inimigo com municao limitada e enfrenta tres niveis de dificuldade. Inclui HUD de acertos e erros, trilha sonora opcional, overlay de video ao final da rodada e interface visual estilo arcade.

## Funcionalidades principais

- Tres niveis de dificuldade (facil, medio, dificil)
- Sistema de municao limitada por rodada
- Tabuleiro configuravel e posicionamento de navios
- HUD com contagem de acertos e erros
- Trilha sonora e efeitos opcionais
- Overlay de video ao encerrar a partida
- Interface responsiva para PC e celular

## Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript
- Netlify

## Como executar

Abra index.html no navegador.

## Deploy / Demonstracao

Publique a pasta raiz no Netlify.
